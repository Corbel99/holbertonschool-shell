# Shell, I/O Redirections and Filters

## Description

This project is part of the Holberton School curriculum.

The goal is to learn and practice:

* Standard input (stdin)
* Standard output (stdout)
* Standard error (stderr)
* Redirections (`>`, `>>`, `<`)
* Pipes (`|`)
* Filters and text processing commands
* Special shell characters

## Learning Objectives

At the end of this project, I should be able to explain:

* What standard input, standard output and standard error are
* How to redirect output to a file
* How to get input from a file
* How to use pipes to connect commands
* How to combine filters to process text
* How to use special shell characters

## Commands Covered

Examples of commands used in this project:

```bash
echo
cat
head
tail
grep
wc
sort
uniq
tr
```

## Requirements

* Ubuntu 20.04 LTS
* All scripts must be executable
* All scripts must start with:

```bash
#!/bin/bash
```
